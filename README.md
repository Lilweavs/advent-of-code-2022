# advent-of-code-2022
Solutions to Advent of Code 2022

## Compiling Code:

With Debugging:
```
g++ -g -Wall -std=c++20 -I ../include main.cpp -o main.exe
```

With optimizations:
```
g++ -O3 -Wall -std=c++20 -I ../include main.cpp -o main.exe
```


To start GDB server
```
gdbserver localhost:65432
```
